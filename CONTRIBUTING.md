# Contributing

Use the project documentation as the main navigation guide:

- [Português](https://github.com/mano-d-ccxxrsj/kutter_api/blob/main/docs/pt-BR/README.md)
- [English](https://github.com/mano-d-ccxxrsj/kutter_api/blob/main/docs/en/README.md)

Before opening or sharing changes, run:

```bash
cargo check
```

```bash
cargo test
```

Keep changes scoped to the feature being worked on, and prefer the existing explicit structure over adding new abstractions early.
